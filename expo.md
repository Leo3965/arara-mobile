> Expo faz o meio de campo para que não seja necessário fazer a config do ambiente,
> porém alguns recuros nativos não são possíveis e o app fica maior

instalando o expo:

- npm install --global expo-cli@4.7.2
- cd orgs-cesta
- yarn start # you can open iOS, Android, or web from here, or run them directly with the commands below.
- yarn android
- yarn ios # requires an iOS device or macOS for access to an iOS simulator
- yarn web

``expo-cli start --tunnel``

expo install expo-font @expo-google-fonts/montserrat
